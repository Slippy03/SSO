# SSO Spring Boot Workshop

## Tools

* Java 21
* Spring Boot 3.3.8
* PostgreSQL 18
* Docker


## Installation
1. Clone this workshop to your computer

```
git clone https://github.com/Slippy03/SSO.git

```
2. Run docker compose
```
docker compose up -d

```

## Run a workshop
```
mvn spring-boot:run

```

## How to test ?
This workshop have swagger dependency that you can test at this url: http://localhost:8080/swagger-ui/index.html#/
